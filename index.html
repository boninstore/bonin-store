Langkah Menghubungkan Google Sheet (Database)

Ikuti langkah ini agar data otomatis tersambung cloud:

  1. Buat Google Sheet baru di Drive Anda.
  2. Klik menu **Ekstensi** > **Apps Script**.
  3. Hapus seluruh kode bawaan Google, paste kode Apps Script terbaru Anda, klik Save.
  4. Klik tombol **Terapkan (Deploy)** > **Penerapan Baru (New Deployment)**.
  5. Pilih tipe **Aplikasi Web (Web App)**, setel aksesnya menjadi **Siapa Saja (Anyone)**, lalu klik Deploy.
  6. Salin link URL Web App tersebut, dan tempelkan ke kolom samping aplikasi ini!
